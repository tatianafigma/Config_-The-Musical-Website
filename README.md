# Config: The Musical Website

A React + Vite website. The original design is available on
[Figma](https://www.figma.com/design/b0i77qChFHxtbQYUsrnZmT/Config--The-Musical-Website).

## Prerequisites

You only need **Node.js** installed (version 18 or newer). It comes with `npm`,
which is all you need to run this project. Download it from
[nodejs.org](https://nodejs.org).

To check that it's installed, open a terminal and run:

```bash
node -v
npm -v
```

## Getting started

1. Open a terminal in this project folder.

2. Install the dependencies (only needed the first time, or after dependencies change):

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the URL shown in the terminal (usually http://localhost:5173) in your browser.
   The page will reload automatically as you edit the code.

## Other commands

- `npm run build` – create an optimized production build in the `dist/` folder.
- `npm run preview` – preview the production build locally.
